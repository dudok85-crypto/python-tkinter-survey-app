# python-tkinter-survey-app
Приложение-анкета с графическим интерфейсом на Python/Tkinter. Пользователь вводит имя, возраст, выбирает интересы (Музыка, Спорт, Книги) и город из списка. После нажатия кнопки программа выводит заполненную анкету в текстовое поле и рисует диаграмму интересов на Canvas. Учебный проект для курсовой работы.
